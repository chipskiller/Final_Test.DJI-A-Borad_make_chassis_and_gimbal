# 初级机器人

## 1. 云台任务（can角度环，spi_IMU，mahony）
- IMU得到角速度、加速度
- mahony姿态解算，得到欧拉角
- can线传输角度补偿，实现自稳（可考虑pid）
  - pitch
  - yaw
- 遥控器数据操控云台

## 2. 底盘任务（can_C610电调、四个麦轮）
- 遥控器得到Vx、Vy
- 麦轮速度解算得到电机速度
- 给四个电机速度值（可用pid控制）
  - 行动模式
  - 静止模式（pid）
  - 旋转模式

## 3. 遥控任务
- 串口中断
- 摇杆数据和拨杆数据
- 通过拨杆决定摇杆控制对象

## 4. FreeMaster
- 创建以上三个任务
